Grade

The letter grade for this homework will be broken down as follows:

Letter Requirement
C
Prototype 3 unique visualizations using Tableau Desktop.
B C-level requirements plus implement 1 of the Tableau prototypes in D3 version 5.
A C-level requirements plus implement 2 of the Tableau prototypes in D3 version 5.

Details on each grade level.

C-Level Functionality

Prototype three different basic visualizations of the dataset using Tableau Desktop. Specifically:The requirments for this level have been met

  • At least 3 different visualizations on Tableau..

  • At least 2 different encodings from the following list:
    • Encode data using points (e.g. scatter or bubble plot)
    • Encode data using bars (e.g. horizontal/vertical bar charts, stacked or grouped bars, small multiples, and histograms)
    • Encode data using area (e.g. area charts, stacked area, small multiples)
    • Encode data using heatmaps or pie charts

  • At least 2 different perspectives of the data.

  • Use of colors to encode data in at least one visualization. This visualization must include a color legend in the static image export.

  • Use titles and axis labels where appropriate to make sure the encoding is clear.

  • For each visualization, include a caption including name and original data source.
B-Level Functionality

Implement one of the Tableau prototypes in D3 version 5. Specifically:The requirments for this level have been met

  • Use the same visualization technique and data encoding as one of Tableau visualizations.

A-Level Functionality

This has the same requirements as the B-level functionality, except for a second Tableau prototype.The requirments for this level have been met
